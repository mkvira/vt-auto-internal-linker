# افزونه لینک داخلی خودکار وردپرس — VT Auto Internal Linker

افزونه **VT Auto Internal Linker** یک ابزار لینک سازی داخلی خودکار برای وردپرس است که بر اساس قوانین کلیدواژه→URL تعریف‌شده توسط شما، لینک‌های داخلی را به‌صورت خودکار در محتوای نوشته‌ها و صفحات درج می‌کند — بدون هیچ تغییری در پایگاه داده.

**نویسنده:** [محمود کاظمی](https://mahmoudkazemi.ir)  
**شرکت:** [ویرا تیم](https://vira-team.com)  
**لایسنس:** GPL-2.0-or-later  
**حداقل نسخه وردپرس:** 6.0  
**حداقل نسخه PHP:** 7.4

---

## ویژگی‌های افزونه لینک سازی داخلی خودکار

- تعریف قوانین کلیدواژه→URL از پیشخوان وردپرس با رابط کاربری ساده
- پشتیبانی از چند کلیدواژه برای هر URL — مدیریت همه کلیدواژه‌ها در یک صفحه
- لینک‌دهی خودکار کلیدواژه‌ها هنگام نمایش محتوا (بدون ذخیره در دیتابیس)
- کنترل‌های دقیق به ازای هر کلیدواژه: اولویت، حداکثر لینک در هر نوشته، سقف کل سایت، آنکر (`#بخش`)، حساسیت به حروف، nofollow، باز شدن در تب جدید
- سقف rule-level: محدودیت تعداد کل لینک به یک URL در هر نوشته (برای همه کلیدواژه‌ها)
- حفاظت از بلاک‌های موجود: هرگز درون تگ‌های `<a>`، `<code>` و `<pre>` لینک نمی‌زند
- جلوگیری از لینک به خود: صفحه‌ای که آدرس آن با URL قانون یکسان است خودکار رد می‌شود
- آمار لینک‌ها: اسکن دستی نشان می‌دهد کدام نوشته‌ها با کدام کلیدواژه لینک شده‌اند
- سازگار با وردپرس مالتی‌سایت
- کاملاً بین‌المللی‌سازی‌شده (فایل `.pot` و ترجمه فارسی همراه افزونه)

---

## نصب افزونه

1. پوشه افزونه را در `/wp-content/plugins/` آپلود کنید
2. افزونه را از بخش **افزونه‌ها** در پیشخوان وردپرس فعال کنید
3. به **تنظیمات → Auto Internal Linker** بروید و اولین قانون لینک‌دهی خود را اضافه کنید

## توسعه

### پیش‌نیازها

- PHP 7.4 یا بالاتر
- WordPress 6.0 یا بالاتر

### راه‌اندازی

```bash
git clone https://github.com/mkvira/vt-auto-internal-linker.git
cd vt-auto-internal-linker
```

### مدل برنچ‌بندی

- `main` — کد پایدار، فقط نسخه‌های منتشرشده
- `develop` — برنچ یکپارچه‌سازی
- `feat/*` / `fix/*` — برنچ‌های کوتاه‌عمر، merge شده از طریق PR به develop

### فرمت commit

```
feat(scope): توضیح کوتاه
fix(scope): توضیح کوتاه
docs: توضیح کوتاه
chore: توضیح کوتاه
```
